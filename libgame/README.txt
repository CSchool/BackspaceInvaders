ON LINUX
--------

install arduino-mk and SFML for emulator.
Flash arduino by running 'make update'
Build emulator by running 'make -C emulator'


ON WINDOWS
----------
Download and extract this archive: https://drive.google.com/open?id=0B89PvMMF3R-XdzdTejdPYW9MSWs
It contains Arduino, MSYS, Python 2.7 and MinGW GNU C/C++.
Copy contents of this folder into 'Source'.
Flash arduino by clicking on 'Update.bat'
Run emulator by clicking on 'Emulator.bat'
